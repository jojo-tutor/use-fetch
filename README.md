# @jtutor/use-fetch
A lightweight react hook that uses fetch API. No additional module dependecies needed!

![Packagist](https://img.shields.io/packagist/l/doctrine/orm.svg)

## Installation
Using NPM:
```shell
$ npm i @jtutor/use-fetch
```

Using Yarn:
```shell
$ yarn add @jtutor/use-fetch
```
